## 变更动机

当前应用虽然已经具备部分文件与导出能力，但核心流程仍不完整且不一致：新建/保存入口尚未统一、PNG 导出尚未真正打通、菜单与操作文案仍夹杂英文。本次变更用于补齐这些体验缺口，让中文用户能够顺畅完成“新建-编辑-保存-导出”全流程。

## 变更内容

- 统一“新建/保存”文件操作，使工具栏动作与快捷键触发行为一致。
- 补齐 PNG 导出链路，使其成为可实际使用的导出格式。
- 将工具栏、导出弹窗及相关操作型菜单文案统一改为中文。
- 确保文案中文化后不影响现有命令逻辑与快捷键映射。

## 能力范围

### 新增能力
- `mindmap-file-operations`: 提供端到端的新建/保存文档流程，并保证 UI 与快捷键行为一致。
- `mindmap-png-export`: 通过现有导出流程支持将当前脑图导出为 PNG。
- `menu-chinese-localization`: 将主要操作命令的菜单/按钮文案统一为中文。

### 修改能力
- 无。

## 影响范围

- 前端受影响范围包括：工具栏、对话框、快捷键提示文案、侧边栏选择器。
- Electron 端受影响范围包括：导出 IPC 与 PNG 保存分支逻辑。
- 需要补充/调整测试，覆盖文件操作行为、导出格式选项以及中文文案展示。
